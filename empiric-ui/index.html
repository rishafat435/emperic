Calculus 101 - Integrals

An integral is the continuous analog of a sum, which is used to calculate areas, volumes, and their generalizations. It is the reverse of a derivative. Here is the standard notation we use for integration in [[KaTeX]].

∫ ab f(x) dx = F(b) - F(a)

Implementation Example

If we want to write a basic program to approximate this using Riemann sums, we can do it in C:

C
#include <stdio.h> double integrate(double (*f)(double), double a, double b, int n) { double step = (b - a) / n; double area = 0.0; for (int i = 0; i < n; i++) { area += f(a + (i + 0.5) * step) * step; } return area; }

Remember to review the [[Fundamental Theorem of Calculus]] note before the exam tomorrow. The connection between integrals and derivatives is paramount.
